# Рунический портал — D&D инструмент для изучения рун

Полностью бесплатное веб-приложение на GitHub Pages + Supabase.

## Быстрый старт

1. Создайте проект на [supabase.com](https://supabase.com) (бесплатный тир)
2. Выполните SQL из файла `supabase-schema.sql` в SQL Editor
3. Скопируйте Project URL и Publishable key в файл `js/config.js`
4. Загрузите все файлы в публичный репозиторий GitHub
5. Включите GitHub Pages (Settings → Pages → main → Save)

Подробная инструкция — в файле README.md из первого архива.
