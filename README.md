# Рунический портал — D&D инструмент для изучения рун

Веб-приложение для игры в D&D, где игроки расшифровывают древний рунический язык.
Сайт работает полностью бесплатно на GitHub Pages + Supabase.

## Возможности

- Создание комнат мастером (защищено паролем)
- Подключение игроков по ID комнаты
- Общий чат с копированием сообщений и реакциями
- Рунический алфавит: игроки делают предположения, мастер видит всё
- Переводчик текста в руны и обратно
- 5 мини-игр для разблокировки рун:
  - Запомни последовательность цветов
  - Найди пары
  - Стаканчики
  - Отсчёт времени
  - Поймай шарики
- Система кодов (по 5 на каждую мини-игру)
- Прогресс игроков виден мастеру в реальном времени

## Быстрый старт (всего 4 шага)

### Шаг 1: Создайте проект на Supabase

1. Зайдите на https://supabase.com и зарегистрируйтесь (бесплатно)
2. Создайте новый проект, запомните пароль БД
3. После создания зайдите в Project Settings > API
4. Скопируйте **Project URL** и **anon public key**

### Шаг 2: Настройте базу данных

1. В проекте Supabase перейдите в **SQL Editor**
2. Откройте файл `supabase-schema.sql` из этого проекта
3. Скопируйте его содержимое и вставьте в SQL Editor
4. Нажмите **Run**

### Шаг 3: Вставьте ключи в конфигурацию

Откройте файл `js/config.js` и замените:

```javascript
const SUPABASE_URL = 'https://YOUR-PROJECT-ID.supabase.co';  // ← Project URL
const SUPABASE_ANON_KEY = 'YOUR-ANON-KEY';                    // ← anon public key
```

### Шаг 4: Загрузите на GitHub Pages

Откройте терминал и выполните:

```
# Клонируйте репозиторий (или создайте новый)
git clone https://github.com/YOUR-USERNAME/YOUR-USERNAME.github.io.git
# Или создайте новый:
# mkdir dnd-runes && cd dnd-runes && git init

# Скопируйте все файлы из папки dnd-runes-site в ваш репозиторий
cp -r dnd-runes-site/* .

# Загрузите на GitHub
git add .
git commit -m "Initial commit: DND Rune Portal"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git push -u origin main

# Включите GitHub Pages в настройках репозитория:
# Settings > Pages > Source: Deploy from a branch > main > / (root) > Save
```

Готово! Сайт будет доступен по адресу `https://YOUR-USERNAME.github.io`

## Использование

1. **Мастер** создаёт комнату (вкладка «Создать комнату»)
2. Мастер получает ID комнаты и сообщает его игрокам
3. **Игроки** подключаются по ID комнаты (вкладка «Подключиться к комнате»)
4. Мастер раздаёт игрокам коды мини-игр (видны в панели мастера)
5. Игроки активируют коды и проходят мини-игры для разблокировки рун
6. Игроки делают предположения о значении рун, мастер видит их прогресс
